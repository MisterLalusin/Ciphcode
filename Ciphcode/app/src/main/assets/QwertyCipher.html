The Qwerty Cipher is the extension to the Substitution Cipher. This cipher works on set of 36 characters only by adding digits and some other symbols to the existing substitution cipher in addition to that changes the mapping sequence used in the substitution cipher. The mapping takes from an alphabet sequence to extended QWERTY keyboard sequence.


source: www.ijritcc.org

Example :

normal   - Qwerty Cipher
encypted - Jvtkzn Eohitk
