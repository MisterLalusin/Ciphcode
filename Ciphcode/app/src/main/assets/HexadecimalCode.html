Hexadecimal numerals are widely used by computer system designers and programmers, as it provides a more human-friendly representation of binary-coded values. Each hexadecimal digit represents four binary digits, also known as a nibble, which is half a byte.

source: wikipedia.org

Example :

normal   - Hexadecimal Code
encypted - 48657861646563696d61
           6c20636f6465
