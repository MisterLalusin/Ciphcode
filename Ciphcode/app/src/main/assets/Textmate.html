Textmate, also known as Code Baby represents its messages with atleast one 11-digit phone number. The first two digits of the code don't have a necessary value while the remaining nine digits represents a maximum of nine character encryption. Numerical value are represented by numbers 500 to 599. A single vowel is represented by number 7 followed by the value of the character, this value is based on where the letter is placed on a 26 letter alphabet. On the other hand, single consonants are a combination of number 8 and the character value. A letter followed by a single vowel is represented by the character value followed by numbers 0 to 4, while a letter followed by a repeating same vowel is represented by the character value followed by numbers 5 to 9. The remaining areas to complete the 11-digit code are filled with numbers 696, which also represents a space between words. Lastly, the end of a sentence or a statement is represented by 666.

Example :

normal   - Textmate
encypted - 09201824820
           09130201696
